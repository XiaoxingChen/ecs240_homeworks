# ecs240_homeworks

## Compress ZIP file
for linux (tab autocompletion available)
```
./build.py pa1-handout
```

for windows
```
python build.py pa1-handout
```
The output zip file is located at `build/pa1-solution.zip`.

## Test

for linux (tab autocompletion available)
```
./build.py pa1-handout --test
```

for windows
```
python build.py pa1-handout --test
```
